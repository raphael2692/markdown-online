What we don't collect

Every conversion tool on __SITE_NAME__ runs entirely client-side, in your browser. Whatever you paste, type, or upload into a tool is processed locally by JavaScript running on your machine — it is never transmitted to this site's server or any third party. There is no upload endpoint for tool content to be sent to.

Hosting and standard web logs

This site is served as static files from a hosting provider (currently Cloudflare Pages), which — like virtually any web host — may record standard technical logs for serving requests (IP address, user agent, requested URL, timestamp) for security and reliability purposes. We do not have access to, and do not use, the content of any tool's input or output, because it never reaches a server.

Cookies and analytics

This site does not currently run any analytics or advertising scripts, and does not set cookies. If that changes — for example, by adding privacy-respecting analytics or an ad network to support hosting costs — this page will be updated first to describe exactly what's added and what data it collects.
