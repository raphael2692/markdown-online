What __SITE_NAME__ collects

Nothing. The tool runs entirely client-side, in your browser. Whatever you paste, type, or import is processed locally by JavaScript running on your machine — it is never transmitted to a server, because this project doesn't run one. There's no account, no analytics, no tracking script, and no cookies set by this site.

Hosting: GitHub Pages

This site is static and open source, served for free by GitHub Pages. We (the project) don't run a server and have no access to any tool's input or output — it never reaches us. GitHub, as the host, is a separate matter: like effectively any web host, it may log basic request metadata (IP address, user agent, requested URL, timestamp) for its own security and operational purposes. That logging is GitHub's, governed by GitHub's Privacy Statement, not something this project configures, sees, or can opt out of on your behalf — we'd rather say that plainly than claim a "zero data" guarantee we can't actually make for GitHub.

Cookies and analytics

This project sets no cookies and runs no analytics or advertising scripts of its own, and has no plans to. If that ever changes, this page will be updated first to say exactly what was added and what it collects.
